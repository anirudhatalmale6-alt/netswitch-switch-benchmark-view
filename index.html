RAM Pressure
—
heap used—
RAM (heap), not ROM (storage), is what fills when a page pulls a big file from a cloud disk into HTML — the whole file lands in the tab's memory. Many tabs + a big pull crosses the per-app RAM limit and Chrome freezes (the "isn't responding" jam). Fix: close pages before the jam, and load big files in chunks, not all at once. The phone camera stalls for the same reason — it needs a large free RAM buffer for each frame; if RAM is near the limit the camera can't get a buffer.
